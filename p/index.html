欧拉函数

定义：对于正整数n，欧拉函数φ(n)的值等于小于n的正整数中与n互质的数目。 
比如φ(5)，小于5的正整数中与5互质的数有1，2，3，4，所以φ(5)＝4.

性质：

  1. 如果n是质数，那么φ(n)=n-1
  2. 如果n=p*q,且p，q均为质数，那么有φ(n)=φ(p)φ(q)=(p-1)(q-1)

:::tips 性质2推导：

1p，2p，3*p……..（q-1)*p与n有公因数p，共有q-1个，同理可以推出与n有公因式q的式子有q-1个，并且这些式子不会出现重复（p，q互质），那么除去这些数剩下的就是于pq互质的数。

∴φ(pq)=(pq-1)-(p-1)-(q-1)=(p-1)*(q-1)

:::

  1. 由性质2，如果p是质数，φ(pk)=pk - pk-1
  2. 对于任意正整数n：

$ \varphi(n)=n*\prod \limits_{i=0}^n{1-\frac{1}{p_i}}（其中p是n的质因数） $

  1. 若a为质数，b mod a＝0，则φ(ab)=aφ(b) (注意与性质2区分）
  2. 结合性质2与性质5，若n=pxqy ,p,q均为质数，则$ \varphi(n)=p^{x-1}q^{y-1}(p-1)*(q-1) $
  3. 所有小于n且与n互质的数的和sum：$ sum=n*\frac{\varphi(n)}{2} $

先证明一条重要的性质：$ \color{blue} 若gcd(n,i)=1,则gcd(n,n-i)=1 $

假设若_gcd(n,i)=1_,则_gcd(n,n-i)=k_，k不为1

那么有_(n-i) mod k =0,n mod k=0_

∴ i mod k =0

所以_gcd(n,i)=k_,与假设矛盾，假设成立。

所以在所有小于n且与n互质的数的集合A中的任意元素a，均有另一个元素b使得a+b=n。接下来不难得到性质7

扩展到不要求互质。

$ \begin{equation}\label{eq2} a^C \equiv\left{ \begin{array}\ a^{c \ mod \ \varphi(m)} &\ \ gcd(a,m)=1\ a^c & \ \ gcd(a,m)\ne1&,c<\varphi(m) \ a^{c \ mod \ \varphi(m) +\varphi(m)} &\ \ gcd(a,m)\neq1&,c \geq \varphi(m) \end{array} \right. \end{equation} $

中国剩余定理

设定一组正整数m1,m2,…….,mk ,俩俩互素，则同余方程组：$ \begin{equation}\label{eq2} \left{ \begin{array}\ x \equiv a_1 (mod \ \ m_1)\ x \equiv a_2 (mod \ \ m_2)\ ……\ x \equiv a_k (mod \ \ m_k)\ \end{array} \right. \end{equation} $有整数解。

且在模$ M=m_1m_2m_3*…*m_k $下的解是唯一的，解为：

$ x \equiv\ (a_1M_1M_1^{-1} + \ a_2M_2M_2^{-1} + …. + \ a_kM_kM_k^{-1} ) \ \ (mod \ \ M) $

简记为：

$ x \equiv\ \sum_{i=1}^ka_iM_iM_i^{-1} \ \ (mod \ \ M),其中M_i=\frac{M}{m_i},M_i^{-1}=M_i^{-1} \ mod\ m_i $

用途：

1.在RSA算法中，可以用于应对一些特殊的情况，如有时出现无法求d的情况

:::color1 m1,m2是互质的俩个正整数，对任意的b1，b2：

       $   \begin{equation}\label{eq2}
\left\{
\begin{array}\\
x \equiv b_1 \ (mod \ m_1)\\

x\equiv b_2 \ (mod \ m_2) \end{array} \right. \end{equation} $

套用公式我们可以得到：

$ x \equiv b_1M_1M_1^{-1}+b_2M_2M_2^{-1} \ \ (mod\ m_1m_2) $

存在第二种表达：

$ x \equiv b_1sm_1+b_2tm_2 \ \ (mod\ m_1m_2)\ \ \ \ \ s,t满足sm_1+tm_2=1 $

在python的第三方库中可以使用gcdtxt函数求解得到s，t

:::

2.正常流程如下：

#此篇以简单介绍为主，具体使用场景因题而异

扩展中国剩余定理：

对于方程组$ \begin{equation}\label{eq2} \left{ \begin{array}\ x \equiv a_1 \ (mod \ m_1)\ x\equiv a_2 \ (mod \ m_2) \end{array} \right. \end{equation} $,记特解为$ x_0 $，则有通解$ x=x_0-k*lcm(m_1,m_2) $,即这个方程组等价于$ x\equiv x_0\ (mod\ lcm(m_1,m_2)) $。

证明：

:::info 原方程组可以改写为：

$ \begin{equation}\label{eq2} \left{ \begin{array}\ x + pm_1=a_1\ x+qm_2=a_2 \end{array} \right. \end{equation} $

于是：

$ pm_1-qm_2=a_1-a_2 $

这个方程的整个解系为：

$ \huge\begin{equation}\label{eq2} \left{ \begin{array}\ p=p_0+k\frac{m_1}{gcd(m_1,m_2)}\ q=q_0-k\frac{m_2}{gcd(m_1,m_2)} \end{array} \right. \end{equation},p_0,q_0是特解 $

:::

利用这个性质我们便可以将模不互素的方程合并为一个方程，再使用中国剩余定理即可解决问题。

原根

定义：若n，a为正整数，且n，a互质，令$ a^d \equiv 1 \ (mod \ n) $,如果使得该式成立的最小的正整数$ \delta(n,a) $=$ d=\varphi(n) $,则a是模n的原根

离散对数

如果对于一个整数b和质数p的一个原根a，可以找到唯一i使得：$ b=a^i\ (mod \ p) $，其中i小于等于p-1成立，则i被称为b的以a为基数的模p的离散对数
